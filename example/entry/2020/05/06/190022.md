# hatenablog-post

Post article of markdown file to hatena-blog.

```sh
$ yarn global add basd4g/hatenablog-post
$ export HATENA_USER=...
$ export HATENA_API_KEY=...
$ export HATENA_BLOG_ID=...

$ hatenablog-post -f path/to/markdown
```
